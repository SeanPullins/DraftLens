# DraftLens Backtest Report

_This file is regenerated by `scripts/train_draftlens_model.py`
(`npm run model:train`). It reports the walk-forward results on your real data;
it is intentionally empty until then so no accuracy is claimed before it is
measured._

After training you'll see:

- the selected outcome target and validation scheme (walk-forward year range)
- overall out-of-sample correlation
- hit / bust rate and average realized value by tier
- correlation + hit rate by position and by class
- biggest correct calls and biggest misses

> numbers are produced.
